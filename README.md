# Эрудит

Классическая настольная игра «Эрудит» с элементами мультсериала «Смешарики».  
Игроки составляют слова из букв, получают очки, голосуют за чужие слова и соревнуются.

## Основные возможности

- 2–4 игрока
- Выбор персонажа (Смешарики) для каждого участника
- Игровое поле 15×15 с премиальными клетками (удвоение/утроение буквы или слова)
- Мешок с фишками (распределение букв и очков соответствует реальному русскому языку)
- Механизм голосования: остальные игроки принимают или отклоняют составленные слова
- Обмен фишек (если в мешке достаточно)
- Пропуск хода и сдача игрока
- Автоматическое определение победителя при окончании игры

## Правила игры

- **Цель** – набрать больше очков, чем соперники.
- **Первый ход** – слово должно проходить через центральную клетку (7, 7).
- **Каждый ход** – можно выкладывать фишки на поле, формируя одно или несколько слов (по горизонтали и/или вертикали).
- После выкладки слово **фиксируется** (кнопка «зафиксировать слово»). Поле подсчитывает очки с учётом премий.
- После **завершения хода** (кнопка «завершить ход») на экран выводятся составленные слова и запускается **голосование** – остальные игроки принимают или отклоняют слова (большинством голосов).
  - Если слово принято – игрок получает очки, добирает фишки до 7, ход переходит следующему.
  - Если отклонено – фишки возвращаются в руку, игрок может продолжить ход или передать ход.
- **Обмен фишек** – можно обменять любое количество фишек, но после обмена ход сразу передаётся следующему игроку.
- **Сдача** – игрок выбывает из игры, его итоговый счёт фиксируется.
- Игра заканчивается, когда остаётся менее 2 активных игроков или при завершении всех ходов. Побеждает игрок с максимальным счётом среди не сдавшихся.

## Технологии

- .NET Framework / .NET Core (Windows Forms)
- C# 10.0
- Модульные тесты (MSTest)

## Требования и запуск

- Операционная система Windows
- Установленная среда разработки (Visual Studio 2022 или новее)
- .NET Desktop Runtime

Для запуска лучше всего клонировать репозиторий в вашу среду разработки.  
Откройте решение `Erudite.sln`, восстановите пакеты (при необходимости) и соберите проект.  
Запуск – через `erudite.exe` или из Visual Studio (F5).

## Структура проекта

Проект состоит из нескольких частей:

- **erudite** – пользовательский интерфейс приложения (Windows Forms).  
  Содержит формы: `InitialForm`, `Choice`, `EruditeForm`, `VoteForm`.

- **ClassLibrary** – основная логика приложения.  
  Ключевые классы: `Tile`, `Player`, `GameController`, `GameBoard`, `BagOfTiles`.

- **TestProject1** – модульные тесты для проверки работы программы.
